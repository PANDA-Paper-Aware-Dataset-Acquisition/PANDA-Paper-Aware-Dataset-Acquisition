## 🐼 Overview

**PANDA (Paper-Aware Dataset Acquisition)** is a framework for automatically discovering datasets from scientific papers.  
It identifies dataset mentions, retrieves official repositories, verifies accessibility, and builds structured metadata to enhance dataset discoverability and research reproducibility.

---

## ✨ Key Features

- 📄 **Paper-Aware Dataset Discovery**  
  Extracts dataset mentions from scientific papers.

- 🔍 **Repository Retrieval**  
  Finds official dataset sources such as GitHub, Hugging Face, Kaggle, Zenodo, and project pages.

- ✅ **Accessibility Verification**  
  Checks whether dataset links are valid, accessible, and usable.

- 🗂️ **Structured Metadata Construction**  
  Builds organized metadata for each dataset, including name, source, URL, availability, and evidence.

- 🔬 **Reproducibility Support**  
  Helps researchers quickly locate datasets required to reproduce scientific results.

## 🚀 Goal

PANDA aims to bridge the gap between **scientific papers** and **real-world datasets** by automating the process of dataset discovery, repository retrieval, accessibility verification, and metadata generation.

---

🐼 Why PANDA?

Finding datasets mentioned in papers is often time-consuming and inconsistent.
PANDA reduces manual search effort by automatically linking papers to their corresponding datasets, making scientific resources easier to find, verify, and reuse.
